# IPL Analytics

A beginner data analysis project on IPL data (2008–2024), built using Python and Pandas.

## What I Did

- Cleaned and merged ball-by-ball delivery data with match data
- Calculated batting stats — strike rate, average, boundary %
- Calculated bowling stats — economy, strike rate, average
- Analyzed venues — runs, sixes, and wickets per match
- Created visualizations to compare top players and venues

## Tech Used

- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Dataset

IPL ball-by-ball dataset (2008–2024) from Kaggle:
🔗 [Dataset Link](https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020)

## Visualizations

- Top 5 batsmen comparison
- Batting form over time (rolling average)
- Venue heatmap
- Batsman impact score ranking
- Season-wise run heatmap
- Bowler efficiency (economy vs wickets)
